Project 01 / Frontend

Cricket Scorecard Web Application

A fully interactive cricket match tracking application built with vanilla JavaScript. Designed to simulate a real scorecard experience for cricket enthusiasts, allowing live score updates, player management, and detailed match statistics — all in the browser with no backend dependency.

Key Features
  • Real-time score updates with dynamic DOM manipulation
  • Per-player statistics: runs, balls faced, strike rate, fours, sixes
  • Innings tracking with run rate & required rate calculation
  • Interactive player selection and batting/bowling UI
  • Responsive layout for mobile scorecard viewing
HTML5 CSS3 JavaScript DOM Manipulation LocalStorage
